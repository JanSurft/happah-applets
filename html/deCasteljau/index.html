De Casteljau algorithm

Grid Control polygon Curve Clear Guide

Description:

In the first step, the algorithm subdivides every edge with a ratio defined by the scrollbar. With the scrollbar's handle in the middle (\(t=0.5\)), every edge is cut in half.
Click on the right arrow to display the next frame!
The newly created points are made by linear-interpolating the corner points. If we connect the new points with lines, we get the new red edges.
We can use the new edges to repeat the process. In the end we only have one new point, which we can't connect any further.
This is the first point on a curve. If you move the scrollbar-handle, the point will move along that curve.
