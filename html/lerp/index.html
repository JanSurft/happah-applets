Linear interpolation

Grid Control polygon Curve Clear Guide

Description:

Pick two control points by clicking inside scene. Click on the right arrow to get to the next frame!
The algorithm will then create a new point \(c\) between the points. The new point divides the red line into two intervals (red/black). The size of each interval is defined by the scrollbar-handle (black square), which you can move along the axis by drag-and-drop!
Let's say your first point is \(a\) and your second point is \(b\) then you get a new point \(c\) with:

\(c = a + t*(b-a)\)

Where \(t\) is the value of your handle within the interval.
Notice how the color of the interval is represented on the scrollbar's axis. This process is called \(linear-interpolation\).
If you move the handle out of the interval \(0-1\) you create a new point by \(linear-extrapolation\).
If you repeat the process of linear interpolation with the newly generated point, you get to De Casteljau's algorithm which is explained in the next section.
